    • Профнастил

    Профнастил – популярный строительный материал, который используется для возведения наружных ограждений, кровельных работ и облицовки стен. Внешне профнастил выглядит довольно привлекательно, обладает рядом разнообразных геометрических форм и цветовых решений. Профнастил устойчив к коррозии, не теряет цвет и хорошо справляется с перепадами температур.

    Виды профнастила:

  • Стеновой профнастил. Стеновые профлисты маркируются буквой «С». Профлист «С» используют для облицовочных работ, для создания разнообразных конструкций (гаражей, ангаров) и для возведения ограждений.

  • Кровельный профнастил (несущий) - более толстый, по сравнению со стеновым. Для таких листов используется маркировка «Н». Благодаря жесткости и прочности, профнастил «Н» выдерживает большие нагрузки.

  • Смешанного типа. Для такого профлиста используется маркировка «НС». Такие листы профнастила могут использоваться как для кровельных, так и для отделочных работ.
